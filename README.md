# Edu Images

Minimal repo to create EduBricks-based material for FAIR asset reuse.

Idea:
  - this repo provides images only

Unless otherwise specified, all images are shared here with CC-BY 4.0 by DataPLANT (https://nfdi4plants.org, https://ror.org/01v7r4v08)

## Repo structure 

### [figures](figures)

- ready-to-use figures for slide decks, texts, websites
- typically 1 figure : 1 slide
- File type: **SHOULD** be .svg
- Lifetime: medium to long

### [logos](logos)

- logos by contributors, partners, softwares
- heavy re-use
- important for corporate, politics, websites and slides (footer, acknowledgement), etc
- Lifetime: long

### [screenshots](screenshots)

- typically screenshots of tools and services
- Lifetime: short to medium

### [shapes](shapes)

- smaller bits (icons, shapes) to reuse for figure creation
- File type: **MUST** be .svg

# Guides

## Working with SVGs

### SVG in powerpoint
  1. import svg (e.g. shape) into powerpoint
  1. right-click + convert to shape
  1. ungroup
  1. edit figure / shape
  1. group
  1. right-click + save as picture -> save as svg

### Drawio

Image files ending in `*drawio.svg` or `*drawio.png` can be opened with
  1. draw.io online: https://draw.io
  2. drawio vscode extension: https://marketplace.visualstudio.com/items?itemName=hediet.vscode-drawio

3. Using SVGs in vscode
  1. SVG Preview: https://marketplace.visualstudio.com/items?itemName=SimonSiefke.svg-preview
  2. SVG Editor: https://marketplace.visualstudio.com/items?itemName=henoc.svgeditor
  3. Change color: svg is a text file, changing colors can be done by text replacement
